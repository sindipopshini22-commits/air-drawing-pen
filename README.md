# Air Drawing Pen

## Description
A camera-based gesture drawing system that allows users to draw in the air using their finger. The system uses MediaPipe Hands for real-time hand tracking and renders drawings on a digital canvas. Inspired by creative interactive tools, it combines computer vision with gesture interaction.

## Features
- Real-time finger tracking
- Draw in the air with your index finger
- Press **Shift** to draw
- Press **Space** to clear the canvas
- Glow brush effect for an aesthetic look
- Works entirely in the browser (no installation needed)

## Tech Stack
- HTML / CSS / JavaScript
- [MediaPipe Hands](https://google.github.io/mediapipe/solutions/hands.html)
- Browser-based camera input
